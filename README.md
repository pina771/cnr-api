# FER, IS Projekt

---

Backend dio projekta Informacijskih sustava. Istraživanje čiste arhitekture i DDD principa.
_CRUD, Auth, Dependency Injection, ORM, Unit testing_

#### Stack:

- Framework : **NestJS**
- Language: **TypeScript**
- ORM : **MikroORM**
- Auth: **JWT**
- Testing: **Jest & NestJS**
